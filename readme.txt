Projekt ktory wyewoluuje - Nazwa PROFLOW

Wykorzystujmy technologie:
Ruby on Rails 8.0,
mySQL
Docker
docker-compose
Bootstrap
Slim
cancancan
devise
ładowanie assetów przez importmaps 
dynamiczne elementy frontend w oparciu o turbo, turbo stream, turbo frames

Niech widocą wartością podczas budowania projektu będzie skalowalność. 

1) Zbudujmy MVP projektu który wstępnie będzie panelem administracyjnym z uytkownikami o rónych poziomach uprawnień. 

2) Czym docelowo będzie projekt jeszcze nie wiadomo. 
3) Na razie chciałbym zbudować estetyczny, czytelny i ładny frontend w oparciu o modyfikację/nadpisywanie standardowych klas, zmiennych bootstrap. Powinien mieć 2 konfigurowalne tryby: ciemny i jasny
4) Kazdą funkcjonalność zanim zaproponujesz kod, omówimy

5) Musimy utworzyć repozytorium git, podpiąć zdalne repo.
6) Mam tez serwer VPS. Po wstępnym przygotowaniu apki zrobimy deploy aplikacji w oparciu o kontener docker i 
7) dodajmy rspec i testy
